TEST 1: SG_BASELINE_DEFAULT_CHECK
Purpose: When ENTITY_ID env var is NOT set, program defaults
         to SG with all canonical SG values.
Setup:   unset ENTITY_ID
Run:     cobc -x PORTVAL.cbl && ./PORTVAL
Expect in working-storage / trace output:
  WS-ENTITY-CODE         = 'SG'
  WS-BASE-CURRENCY       = 'SGD'
  WS-ENTITY-LOCALE       = 'en_SG'
  WS-REGULATOR-CODE      = 'MAS'
  WS-BOOKING-CENTRE      = 'Singapore'
  WS-FEE-BPS             = 50
  WS-LARGE-POS-THRESHOLD = 250000
  WS-SUITABILITY-FW      = 'MAS_FAA_2002'
  WS-ENT-KV-VAULT        = 'wealth-sg'
  DB secret ref pattern  = '${KEYVAULT:wealth-sg-db-connection-string}'
Verdict: PASS if all values match exactly.
